Copy a Channel Preset on the X32

When a channel has EQ, compression, and processing settings that are dialed in exactly right, you can copy that entire configuration to another channel in seconds — without rebuilding it from scratch.

Time: ~2 minutes
Difficulty: Beginner
When: Soundcheck or pre-service setup
What is a channel preset? On the X32, every channel's processing settings — EQ curve, gate, compressor, FX sends — can be saved as a preset and recalled on any other channel. This is how you quickly give a second vocalist the same starting EQ as the first, or restore a channel after an accidental change.
What Gets Copied
✓ Included in the copy
4-band parametric EQ settings
High-pass and low-pass filter settings
Gate — threshold, range, attack, hold, release
Compressor — threshold, ratio, attack, release, knee
Makeup gain
FX send assignments
Channel color and name
✗ Not included in the copy
Fader position
Preamp gain (analog gain)
Phantom power (+48V) state
Bus send levels
Routing assignment (what input feeds this channel)
Mute state
Step-by-Step
1
Select the source channel — the one with settings you want to copy
Press the SELECT button on the channel strip whose settings you want to copy. The SELECT button should illuminate. Confirm the correct channel appears in the LCD display at the top of the board.
Press SELECT on the source channel
2
Open the Library tab on the main LCD
Press the LIBRARY button in the navigation tab column on the right side of the main LCD section. The Library screen gives you access to preset save and recall functions for channels, effects, and scenes.
Press LIBRARY on the navigation tabs
3
Make sure Channel is selected as the preset type
The Library tab shows different preset types — Channel, Effects, Scenes. Confirm you are in Channel preset mode. The encoder buttons beneath the LCD let you navigate between types.
Select: Channel (not Effects or Scene)
Note: The channel library stores individual channel processing settings. Effects libraries store FX rack configurations. Make sure you are in the right library before saving.
4
Save the source channel as a preset
Use the encoder to navigate to an empty preset slot. Press SAVE. The board will prompt you to name the preset — name it something descriptive like "Lead Vocal" or "Male BGV." Confirm the save.
Navigate to empty slot → SAVE → name it → confirm
Tip: Use a name that describes the source, not the channel number. Channel numbers change between setups; descriptive names are always useful.
5
Select the destination channel — where you want to copy the settings
Press SELECT on the destination channel strip — the channel you want to receive the copied settings. Confirm the correct channel appears in the LCD.
Press SELECT on the destination channel
Important: Double-check which channel is selected before loading. Loading a preset overwrites the destination channel's processing settings immediately. There is limited undo.
6
Load the preset onto the destination channel
In the Library tab, navigate to the preset you just saved. Press LOAD. The board will confirm. Press YES. The destination channel's EQ, gate, and compressor settings are now identical to the source channel.
Navigate to preset → LOAD → YES
7
Verify and adjust the destination channel
The preset is now loaded. Confirm the EQ and dynamics look correct on the destination channel's home screen. Then reset the preamp gain, phantom power state (if needed), and fader position — these are not copied and must be set independently.
Check EQ screen → set gain → confirm phantom power → set fader
Remember: Preamp gain is the first thing you set on any channel — including one that just received a preset. The preset gives you the processing starting point; you still need to set gain for the actual microphone and source.
When to Use This
Two vocalists with similar voices
Dial in one vocal channel, save it, load it on the second. Then adjust each independently for the individual voice.
Restoring a channel after accidental changes
If a channel's EQ or compression was accidentally changed during a service, load its saved preset to restore the baseline settings instantly.
Building a library of starting points
Over time, save presets for "Kick Drum," "Snare," "Lead Vocal," "Acoustic Guitar." These become reliable starting points that speed up soundcheck every service.
